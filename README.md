# Expense-Tracker
Автор: [Илиади Станислав]

Описание программы
«Expense Tracker» — это простое и удобное GUI-приложение, предназначенное для учета личных финансов. Программа позволяет пользователю быстро вносить данные о своих тратах, сохраняя их в локальный JSON-файл для последующего анализа. Основные функции включают добавление записей, фильтрацию расходов по категориям или датам, а также расчет общей суммы трат.

Как использовать
Запуск: Убедитесь, что у вас установлен Python, и запустите файл Expense Tracker.py.

Добавление расхода:

Введите сумму (положительное число).

Укажите категорию (например, "Еда", "Транспорт").

Выберите дату в формате ГГГГ-ММ-ДД (по умолчанию подставляется текущая дата).

Нажмите кнопку «Добавить расход».

Фильтрация: Введите значение в поле фильтра (название категории или конкретную дату) и нажмите кнопку «Фильтр». Для возврата ко всему списку нажмите «Сброс».

Подсчет: Нажмите кнопку «Подсчет общей суммы», чтобы увидеть итоговые траты.

Примеры использования
Добавление: Сумма 1500, Категория Развлечения, Дата 2026-04-29. Нажмите «Добавить расход». Запись появится в таблице.

Фильтрация: Введите в поле фильтра слово Развлечения и нажмите «Фильтр» — таблица отобразит только соответствующие записи.

Подсчет: Нажмите кнопку «Подсчет общей суммы» для получения суммы всех внесенных расходов за всё время.

Технические детали
Данные: Все данные сохраняются автоматически в файл data.json.

Контроль версий: В проекте используется Git. Файл .gitignore настроен на исключение файла базы данных data.json и системных папок Python, чтобы не засорять репозиторий.
